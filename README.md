# Floppy-Browser
The floppy net desktop web browser

## How it works?
- Use a "Start up disk" to connect to the floppy net
- You only need drag and drop the start_net.html file in the browser window
- And click in the connect button

## Platafroms
- [x] Windows
- [ ] MacOS
- [ ] Linux distros
